STEP 2: ADD MINIMUM ENVIRONMENT VARIABLES (3 minutes)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Go to: https://vercel.com/hostiliann/collab-connect/settings/environment-  │
│        variables                                                             │
│                                                                              │
│ Add these 4 variables (check all: Production, Preview, Development):        │
│                                                                              │
│ 1. DATABASE_URL                                                              │
│    Value: postgresql://user:pass@host:5432/collab_connect                   │
│    (Get from Vercel Postgres: vercel.com/dashboard/stores)                  │
│                                                                              │
│ 2. NEXTAUTH_SECRET                                                           │
│    Value: (generate with: openssl rand -base64 32)                          │
│    Or use deployment manager option #2                                      │
│                                                                              │
│ 3. NEXTAUTH_URL                                                              │
│    Value: https://hostilian.org                                             │
│                                                                              │
│ 4. UPSTASH_REDIS_REST_URL                                                    │
│    Value: (sign up at upstash.com - 2 min, free tier)                      │
│                                                                              │
│ 5. UPSTASH_REDIS_REST_TOKEN                                                  │
│    Value: (from upstash.com after creating database)                        │
└─────────────────────────────────────────────────────────────────────────────┘

STEP 3: REDEPLOY
┌─────────────────────────────────────────────────────────────────────────────┐
│ After adding variables:                                                      │
│ • Go to Vercel Dashboard → Deployments                                      │

│                                                                              │
│ OR just push:                                                                │
│   git commit --allow-empty -m "chore: trigger deployment"                   │
